unit
unit is the property used to define length value such as height and width
there are two types of length units
  • absolute unit
  • relative unit
  1. absolute unit:

    2. this are fixed values and it does't depend upon any refference like parent element so it will give exat size what we include

    cm centimeters
    mm millimeters
    px pixels
    in inches
    pt points
    pc picas

  2. relative unit:

    3. it will reffer to the parent element which is relative to the current element

    em element
    ex x-height
    ch
    rem
    vw view port
    vh viewport height
    v min minimum
    v max maximum
    % percentage
